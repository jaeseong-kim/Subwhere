## 프로젝트 소개
    내가 탄 지하철을 선택해서 지하철이 현재 무슨 역에 있는지 알 수 있는 서비스입니다. 
    지하철을 탔을 때 차내 전광판 광고로 인해 역명이 안보이거나 플랫폼의 역명이 차체로 인해 안 보여서 개발하게 되었습니다.
